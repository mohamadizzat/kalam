## 6. Social Scheduler Cron

```
┌──────────────────────────────────────────────────────────────────┐
│              SCHEDULER (every 60 seconds)                        │
└──────────────────────────────────────────────────────────────────┘

    ┌─────────────────────────────────────────┐
    │  SELECT * FROM social_posts             │
    │  WHERE status = 'scheduled'             │
    │  AND scheduled_at <= NOW()              │
    │  ORDER BY scheduled_at ASC             │
    │  LIMIT 5                                │
    └────────────────────┬────────────────────┘
                         │
                    ┌────▼────┐
                    │ Empty?  │
                    ├────┬────┤

               └───────┘   │

                    ┌──────▼──────┐

                    │ post:       │
                    └──────┬──────┘
                           │
                    ┌──────▼──────┐
                    │ 1. Lock     │
                    │   (status → │
                    │  publishing)│
```
