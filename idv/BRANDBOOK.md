## I. A MARCA

### Essencia

**Kalam** (كلام) significa "A Palavra" em arabe. E um movimento de completude espiritual que apresenta a mensagem do Alcorao a brasileiros que nunca a ouviram — sem pressao, sem oposicao, sem superioridade. Apenas beleza, verdade e reconhecimento.

### Proposito

> "Tudo que voce acredita e verdade. Mas falta o capitulo final."

Kalam existe porque levar a Palavra de Deus a quem nunca ouviu e responsabilidade pessoal. Nao e organizacao religiosa. E movimento de completude.

### Posicionamento

Kalam ocupa um espaco unico: **premium spiritual content**. Compete com Masterclass, Apple TV+, e grandes plataformas de conteudo — nao com sites de mesquita ou apps islamicos tradicionais. A estetica e parte da Da'wah (convite).

### Publico-Alvo

Homem brasileiro, 20-35 anos. Segue Pablo Marcal ou similares. Consome conteudo sobre mentalidade, disciplina, empreendedorismo. Acredita em Deus com conviccao, cita a Biblia. NUNCA leu o Alcorao, NUNCA conheceu muculmano praticante.

| **Direto** | Fala sem rodeios | Grosseiro ou agressivo |

- **Contraste minimo:** 4.5:1 pra texto body, 3:1 pra texto grande (WCAG AA)
- **Dark mode e FORCADO** — nao existe light mode no Kalam

## VI. FOTOGRAFIA & IMAGEM

### 6.1 Direcao Fotografica

A fotografia do Kalam segue uma estetica **cinematografica contemplativa**. Cada imagem deve parecer um frame de um filme premiado — nao uma foto de banco de imagens.

**Atmosferas aprovadas:**
- Deserto ao entardecer — luz dourada, vastidao, silencio
- Noite estrelada — cosmo, infinito, meditacao
- Calligrafia close-up — tinta dourada, papel escuro, artesanal
- Arquitetura geometrica — padroes islamicos, luz filtrada, sombras
- Natureza contemplativa — montanha, oceano, floresta (sem pessoas)
- Cidade a noite — luzes bokeh, reflexoes, solidao urbana

**Tratamento:**
- **Temperatura:** Quente (shadows warm, highlights gold)
- **Contraste:** Alto, com sombras profundas
- **Saturacao:** Reduzida (-20% do natural), exceto dourados
- **Grain:** Sutil, cinematografico (emula filme)
- **Vignetagem:** Sutil, direciona foco ao centro

### 6.2 Proibicoes de Imagem

1. **NUNCA** usar fotos de mesquita em hero/landing/capa
2. **NUNCA** usar fotos de pessoas rezando como elemento principal
3. **NUNCA** usar stock photos genericas (handshake, pessoas sorrindo)
4. **NUNCA** usar imagens com verde como cor dominante
5. **NUNCA** usar crescente islamico como elemento visual
6. **NUNCA** usar fotos low-res ou com marcas d'agua
7. **NUNCA** usar imagens que parecem "conteudo islamico generico"

### 6.3 Regra de Gradualidade

Elementos islamicos/arabes entram **GRADUALMENTE** conforme o usuario aprofunda:
- **Home/Landing:** Zero simbolos — apenas estetica premium, frases universais
- **Estudos iniciais:** Caligrafia sutil, padroes geometricos
- **Estudos intermediarios:** Versiculos em arabe, nomes de surahs
- **Estudos avancados:** Conteudo teologico completo, terminologia islamica

## VIII. MOTION & ANIMACAO

### 8.1 Principios de Motion

O movimento no Kalam reflete seus valores: **contemplativo, proposital, nunca apressado**.

| Principio | Descricao | Referencia |
|-----------|-----------|------------|
| **Intencional** | Cada animacao tem proposito — guiar, revelar, ou enfatizar | Apple product pages |
| **Fluido** | Transicoes suaves, sem saltos — ease-out predomina | Masterclass UI |
| **Contemplativo** | Velocidade moderada-lenta — convida a pausar | Calm app |
| **Revelador** | Conteudo aparece como se estivesse sendo revelado | Netflix intro |

### 8.2 Curvas de Easing

```css
/* Padrao — maioria dos elementos */
--ease-default: cubic-bezier(0.25, 0.1, 0.25, 1.0);

/* Entrada — elementos aparecendo */
--ease-in: cubic-bezier(0.0, 0.0, 0.2, 1.0);

/* Saida — elementos saindo */
--ease-out: cubic-bezier(0.4, 0.0, 1.0, 1.0);

/* Enfase — bounce sutil, destaque */
--ease-emphasis: cubic-bezier(0.175, 0.885, 0.32, 1.275);

/* Dramatico — reveals importantes */
--ease-dramatic: cubic-bezier(0.16, 1, 0.3, 1);
```

### 8.3 Duracoes

| Tipo | Duracao | Uso |
|------|---------|-----|
| **Micro** | 150-200ms | Hover, focus, toggles |
| **Standard** | 300-400ms | Fade in/out, slides |
| **Emphasis** | 500-700ms | Reveals de conteudo, destaques |
| **Dramatic** | 800-1200ms | Hero animations, logo reveal |
| **Cinematic** | 1500-3000ms | Transicoes de pagina, splash |

### 8.4 Padroes de Animacao

**Fade Up (padrao):**
```typescript
// Framer Motion
const fadeUp = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.6, ease: [0.25, 0.1, 0.25, 1.0] }
};
```

**Stagger (listas/grids):**
```typescript
const stagger = {
  animate: { transition: { staggerChildren: 0.1 } }
};
```

**Gold Glow Pulse (destaques):**
```typescript
const goldPulse = {
  animate: {
    boxShadow: [
      '0 0 0px rgba(201, 168, 76, 0)',
      '0 0 20px rgba(201, 168, 76, 0.3)',
      '0 0 0px rgba(201, 168, 76, 0)'
    ],
    transition: { duration: 2, repeat: Infinity }
  }
};
```

**Scroll Reveal (Lenis):**
```typescript
// Elementos revelam ao scroll com parallax sutil
const scrollReveal = {
  initial: { opacity: 0, y: 40 },
  whileInView: { opacity: 1, y: 0 },
  viewport: { once: true, margin: '-100px' },
  transition: { duration: 0.8, ease: [0.16, 1, 0.3, 1] }
};
```

## XIII. DO'S AND DON'TS

### DO (Faca)

- Usar fundo escuro (#0A0A0A) como padrao
- Aplicar gold (#C9A84C) com moderacao — menos e mais
- Usar caligrafia arabe como elemento estetico
- Manter espacamento generoso — respiro visual
- Usar fotografia cinematografica de alta qualidade
- Seguir a regra de gradualidade (elementos islamicos)
- Manter consistencia de tom em todas as plataformas
- Testar contraste WCAG AA em todo texto
- Usar Framer Motion pra animacoes fluidas
- Priorizar mobile-first em todo design

### DON'T (Nao faca)

- Usar fundo verde como cor dominante
- Colocar crescente islamico em qualquer material
- Usar fotos de mesquita em hero/capa
- Usar preto puro (#000000) ou branco puro (#FFFFFF)
- Colocar "Islam" ou "Muculmano" em titulos ou home
- Usar auto-play de audio ou pop-ups
- Usar stock photos genericas
- Misturar mais de 2 familias tipograficas na mesma composicao
- Usar efeitos 3D, drop shadows pesados, ou gradientes coloridos
- Comprometer qualidade por velocidade — se nao compete com premium secular, nao publica
